# TeX Sauce Maker

数式画像からTeX Sauce形式のコードブロックや解説を生成するWebアプリケーションです。

## バージョン履歴

### v1.8.8
- Obsidian用Markdownで、`### 解答` を2回目以降の ` ```latex ` コードブロックの直前へ必ず挿入するように変更。
- Service Workerのキャッシュを更新し、`app.js` のバージョンクエリと更新確認を追加。
- Webアプリが古いJavaScriptを保持し、表示バージョンが更新されない問題を修正。

### v1.8.0〜v1.8.7（復元記録）

この期間は版ごとの変更記録が残っていないため、現存するソースコードから確認できる追加・変更点をまとめて保管しています。各項目を個別の版番号へ割り当てることはしていません。

- 生成結果を直接編集できるCodeMirrorベースのLaTeXエディターを追加。
- PDFの1ページ目を画像として読み込む機能を追加。
- Clear Maker 2から画像とTikZコードを受け取る連携機能を追加。
- 受信したTikZコードを生成時の指示へ組み込む処理を追加。
- 画像のクリップボード貼り付けと複数画像の取り扱いに対応。
- 追加指示を与えて再生成する機能を追加。
- 既定の生成モデルを `gemini-3.1-pro-preview` に更新。
- Obsidian用Markdownに問題・解答のLaTeXコードブロックとPDF埋め込みを出力する処理を追加・改善。
- コードフェンスの一部が欠けた生成結果を補正し、問題と解答を分離する処理を追加。
- 難易度判定で解答内容、タグ、場合分け、計算量、論証の長さ、発想の必要性を参照するよう改善。

### v1.7.9
